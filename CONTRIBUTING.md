# Contributing to Astra Language (astra_lang)

Thank you for your interest in contributing to **Astra Language**, the intent-driven, AGI-ready programming language core!

Your contributions help build a powerful language that empowers artificial general intelligence with motive and intent.

---

## How to Contribute

### Reporting Issues

- Please use the [GitHub Issues](https://github.com/gd2bk1ng/astra_lang/issues) to report bugs, suggest features, or request enhancements.
- Provide clear, concise descriptions and steps to reproduce issues.

### Development Setup

1. **Clone the repository:**

```bash

git clone https://github.com/gd2bk1ng/astra_lang.git
cd astra_lang

```

Build and test:


```bash

cargo build
cargo test

```
Make your changes on a new branch named descriptively, e.g., feature/parser-enhancement.
